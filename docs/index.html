Como usar

  1. Abra UNO_patched.html diretamente no navegador (duplo clique) ou sirva com um HTTP server local.
  2. Abra DualTube_patched.html para testar playlists e o chat integrado.
  3. Se for usar os recursos de Chat IA, configure sua chave do OpenRouter e modelo pelo UI.

OBS: Alguns recursos (voz/microfone, permissões e Picture-in-Picture) podem pedir autorização do navegador.
